## 목차 섹션 (순서대로)

### 파트 A: UI/UX 디자인 도구

#### Figma — UI 설계 표준
- Figma vs Canva 언제 뭘 쓰나 (표로 명확히)
  - Figma: 웹/앱 UI 설계, 개발 핸드오프, 컴포넌트 시스템
  - Canva: 마케팅 콘텐츠, SNS, 빠른 생산
- Figma AI (Gemini 통합): Make Design, 레이어 자동 정리, 시각적 검색
- Figma Make: 텍스트로 UI 프로토타입 생성
- Figma Dev Mode: 디자인 → 코드 자동 변환 (Claude Code 핸드오프)
- 뉴스레터 서비스 랜딩페이지 Figma 설계 예시

#### Canva Magic Studio — 비디자이너 최강 도구
- Magic Design: 텍스트로 첫 초안 생성
- Magic Resize: 플랫폼별 자동 리사이징 (인스타 → 유튜브 썸네일 → 링크드인)
- Content Planner: 제작 후 예약 발행까지
- Brand Kit: 로고·색상·폰트 일괄 적용
- AI 슬롭 안티패턴 (UI_GUIDE.md 내용 활용)

### 파트 B: 이미지 생성 실전 마스터

#### 이미지 프롬프트 6요소
주제(Subject) / 스타일(Style) / 조명(Lighting) / 구도(Composition) / 색감(Color) / 품질(Quality)

#### 도구별 특성과 사용법
- Flux 2 (bfl.ai): 사실적 이미지, 텍스트 렌더링, 포토리얼리즘 — 뉴스레터 썸네일
- Midjourney v7: 아트·일러스트 — 브랜드 이미지
- Google AI Studio (Imagen 3): 한국어 프롬프트, Gemini 통합 — 무료 시작
- Adobe Firefly: 상업적 라이선스 안전 — 유료 콘텐츠

#### 실전 워크플로우
뉴스레터 서비스 기준:
1. 헤더 이미지 생성 (Flux 2)
2. SNS 카드뉴스 5장 (Canva Magic Design)
3. 유튜브 썸네일 (Canva + Flux 2 조합)
4. OG 이미지 (소셜 공유 시 보이는 이미지)

### 파트 C: 음악·오디오 생성 실전

- Suno v5: 가사+멜로디+보컬 한 번에, 한국어 노래 가능
- Udio: 음질 중심 BGM
- ElevenLabs: 내 목소리 클로닝 → 유튜브 내레이션 자동화
- NotebookLM: 문서 → 팟캐스트 자동 변환

실전: 뉴스레터 소개 BGM 30초 제작 (Suno)

### 파트 D: 영상 생성 실전

- Veo 3.1 (Google): 영화급 영상 + 음향 자동, Google AI Studio 접근
- Sora 2 (OpenAI): 창의적 영상
- Runway Gen-4: 생성 + 편집 통합
- CapCut AI: 숏폼 특화, 자막 자동

실전 워크플로우 — 60초 숏폼 하루 완성:
대본(Claude) → 이미지(Flux 2) → 보이스(ElevenLabs) → 편집(CapCut) → BGM(Suno)

### 파트 E: 4채널 콘텐츠 재활용 전략

1편 → 워드프레스 블로그 / 인스타그램 / 유튜브 쇼츠 / 스레드 동시 배포
- Canva Magic Resize로 플랫폼별 자동 변환
- 뉴스레터 콘텐츠 → 4채널 배포 실전 예시
- 저작권·상업적 사용 주의사항
