DESIGN PRINCIPLES

커리큘럼 디자인 원칙

9 Stage 모두 같은 원칙으로 설계되었습니다. 한 Stage를 익히면 다음 Stage가 자연스럽게 손에 익습니다.

🎯
One Project 원칙
9 Stage 전체를 AI 뉴스레터 자동화 서비스 하나로 관통합니다. 추상적 예시 없이 한 서비스의 진화 과정으로 학습합니다.
📦
Stage 연결 고리
각 Stage는 명시적으로 "이전에서 받아온 것"과 "다음에 넘겨줄 것"을 정의합니다. 학습이 흩어지지 않습니다.
✅
산출물 미션
읽기만 해서는 안 됩니다. 각 Stage 말미에 2~3개의 실제 산출물을 만들어야 다음으로 넘어갑니다. 총 25개.
🤖
AI 튜터 내장
각 Stage HTML에 Claude API 직결 AI 튜터가 들어 있습니다. 5개 추천 질문 + 자유 입력. Stage 컨텍스트로 맞춤 답변.
🎨
색상 = 진도
9 Stage는 9가지 색상(파랑→그린→보라→핑크→오렌지→틸→옐로우→시안→골드). 진도 dot으로 항상 위치를 시각화합니다.
📜
독립 실행 단일 파일
각 Stage는 완전 독립 단일 HTML 파일. 빌드 도구 X, 의존성 X. 브라우저에서 열면 바로 학습 시작.
