Figma — UI 설계 표준

웹·앱 화면 설계의 사실상 표준. 2026년 Gemini가 통합되면서 더 강해졌습니다

Figma는 "사이트나 앱의 모습을 그리는 곳"입니다. 손으로 그리지 않습니다. Figma AI(Gemini 통합)가 텍스트로 첫 화면을 만들고, Figma Make가 프로토타입을 자동 생성합니다. 그리고 Figma Dev Mode가 그 디자인을 코드로 바꿔 Claude Code로 핸드오프합니다.

⚖️ Figma vs Canva — 언제 뭘 쓰나

Canva Magic Studio — 비디자이너 최강 도구

디자인을 못 해도 일정한 품질의 콘텐츠를 매일 양산할 수 있게 해주는 마법

Canva의 진짜 가치는 템플릿이 아니라 Magic Studio입니다. AI가 디자인을 만들고, 한 번에 모든 플랫폼 사이즈로 변환하고, 브랜드 일관성을 자동으로 유지합니다. 1인 창업가의 콘텐츠 제작 시간을 1/10로 줄이는 도구.

Magic 기능역할뉴스레터 서비스 활용
Magic Design 텍스트로 디자인 첫 초안 생성 "AI 뉴스레터 인스타 카드 — 미니멀, 그린 포인트, 텍스트 강조"
Magic Resize 한 디자인 → 모든 플랫폼 사이즈로 즉시 변환 인스타 1080² → 유튜브 썸네일 1280×720 → 링크드인 1200×627 한 번에
Magic Write 본문·카피 자동 생성 SNS 카드뉴스의 5장 카피를 한 번에 생성
Brand Kit 로고·컬러·폰트 등록 → 모든 디자인에 일괄 적용 그린 #10b981 + 로고 + 폰트 등록하면 모든 콘텐츠 자동 통일
Content Planner 제작 → SNS 예약 발행 일주일 분량 콘텐츠를 예약해두고 자동 발행
⚠️ AI 슬롭(Slop) 안티패턴 — 절대 피할 것
1. 기본 템플릿 그대로: 누구나 알아봄. 신뢰도 폭락. 최소한 컬러·폰트는 내 Brand Kit으로.
2. AI 생성 이미지 표시 없이 사용: 사람 얼굴은 특히 위험. 명확히 "AI generated" 또는 실사용.
3. 모든 콘텐츠를 같은 패턴으로 양산: 알고리즘이 "스팸"으로 분류. 일주일에 1~2번은 직접 손으로 다듬기.
4. 카피가 비슷한 톤: ChatGPT 특유의 "여러분, ~~함께 ~~해요!" 톤 → 즉시 알아챔. UI_GUIDE.md에 금지 표현 등록.

이미지 프롬프트 6요소

"멋있게" 같은 추상어 대신, 6개 항목을 명확히 — 결과물 품질이 10배 차이

AI 이미지가 별로인 이유는 모델이 약해서가 아니라 프롬프트가 모호해서입니다. 6요소를 다 채우면 어떤 모델이라도 결과가 좋아집니다. 외워두세요.

01 · SUBJECT
주제
무엇이 보이는가
예: "노트북을 보며 커피를 마시는 30대 남성"
02 · STYLE
스타일
어떤 미감인가
예: "포토리얼리스틱", "미니멀 일러스트", "1990 잡지 스타일"
03 · LIGHTING
조명
빛이 어떻게 떨어지나
예: "자연광", "골든아워", "뒤에서 비치는 림 라이트"
04 · COMPOSITION
구도
카메라 위치와 앵글
예: "정면 클로즈업", "오버숄더", "탑뷰 플랫레이"
05 · COLOR
색감
전체 톤·팔레트
예: "따뜻한 베이지+그린", "쿨톤 블루+화이트", "모노크롬"
06 · QUALITY
품질
해상도·디테일
예: "8K, 샤프 디테일", "필름 그레인", "85mm f/1.4 보케"
📰 6요소를 적용한 뉴스레터 헤더 이미지 프롬프트
나쁜 예: "뉴스레터 멋있는 이미지" → AI가 갈피를 못 잡음
좋은 예: "노트북으로 뉴스레터를 읽는 30대 남성[주제], 미니멀 일러스트 스타일[스타일], 자연광 위에서[조명], 측면 미디엄 샷[구도], 다크 그린 + 베이지 팔레트[색감], 깔끔한 벡터 디테일[품질]"

6요소 형식만 지키면 Flux 2든 Midjourney든 일관된 품질이 나옵니다.
